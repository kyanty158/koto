📝 すぐメモ

1秒で起動、その瞬間を逃さない。
思いついた瞬間にメモ、未来の自分にリマインド。

⚡️ 爆速起動

アプリを開いた瞬間、キーボードが即座に表示。入力欄をタップする手間すらありません。起動から入力開始まで約1秒。

🔔 直感リマインダー

「10分後」「今夜」「明日朝」などのプリセットから選ぶだけ。面倒な時刻設定は不要。未来の自分へメッセージを送れます。

🛡️ 完全オフライン

データは全て端末内に保存。通信待ちゼロで、機内モードでも地下鉄でもサクサク動きます。

✨ シンプル操作

「破棄」「保存」「保存＋通知」の3ボタンだけで操作完結。2本指タップでUndo（元に戻す）も可能。
